# 🚀 LiteForm

**LiteForm** 是一款极致轻量、高性能的表单收集工具。它专注于“清理、快速、简单”，采用单文件 Go 语言编写，支持中英双语，提供多链接管理功能。

---

## ✨ 核心特性

### 1. 交互与视觉
*   **智能输入体验**：所有输入框均为自适应 `textarea`，随内容自动扩容，输入再多也一目了然。
*   **双语界面**：一键切换中/英文界面（含管理后台、表单、错误提示及导出文件）。

### 2. 分发控制
*   **多链接分发**：支持生成无限个独立的分享链接，可针对不同渠道设置不同备注。
*   **安全屏障**：支持为每个链接设置独立的**访问密码**。
*   **时效管理**：支持设置链接**过期日期**，过期自动失效。
*   **隐私保护**：根目录 `/` 默认隐藏，仅允许通过特定分享链接访问。

### 3. 管理后台
*   **动态字段配置**：随时在线修改表单字段名称（支持中英双套字段）。
*   **实时数据查看**：后台实时展示提交详情，并自动标注数据来源（通过哪个链接提交）。
*   **一键数据导出**：支持带 UTF-8 BOM 的 CSV 导出，确保 Excel 打开不乱码，包含来源溯源。
*   **物理清空**：支持一键物理清除所有数据并执行 `VACUUM` 压缩数据库体积。

## 🛠 快速搭建

只需一行命令，即可在你的服务器上启动 **LiteForm**。

```bash
docker run -d \
  --name liteform \
  -p 8255:8255 \
  -v $(pwd)/data:/app/data \
  -e ADMIN_USER=admin \
  -e ADMIN_PASS=你的管理员密码 \
  -e TZ=Asia/Shanghai \
  --restart always \
  lockcp/liteform:latest
```

### 环境变量说明
| 变量名 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `ADMIN_USER` | `admin` | 管理员用户名 |
| `ADMIN_PASS` | (必填) | 管理员登录密码 |
| `PORT` | `8255` | 容器内部运行端口 |
| `TZ` | `Asia/Shanghai` | 时区设置 |

## 📖 使用指南

1.  **访问后台**：打开 `http://你的IP:8255/admin`，使用设置的管理员账号登录。
2.  **配置字段**：在后台“字段配置”区块，输入你想收集的信息项（逗号分隔）。
3.  **生成链接**：在“生成分享链接”区块，输入备注名、密码（可选）和过期时间（可选）。
4.  **分发表单**：复制生成的链接发送给用户即可。
5.  **查看数据**：数据会实时显示在后台表格中，你可以随时点击“导出 CSV”进行下载。

## ⚡️ 技术规格

*   **后端**：原生 Go + `net/http` (无重型框架，极速响应)
*   **数据库**：SQLite3 (单文件存储，迁移极简)
*   **前端**：原生 HTML5 + CSS3 (Vanilla CSS) + JavaScript
*   **部署**：Docker / Docker Compose (体积仅约 15MB)
